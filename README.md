### 🌙 Baraka &nbsp;·&nbsp; [GitHub](https://github.com/Ammarahmed1263/Baraka)

> *"إنما الأعمال بالنيات"* — actions are only by their intentions.

An offline-first Islamic habit tracker that turns the prophetic concept of *niyyah* (intention) into a daily practice. Built without a backend — every byte lives on-device via MMKV. Features role-based personalization (Student, Parent, Spouse), full bilingual Arabic/English RTL, streak tracking, push notifications, and Sentry crash monitoring. Shipped to closed beta on Google Play.

```
React Native · Expo SDK 54 · Expo Router v4 · Zustand · MMKV · Reanimated · i18next · Sentry
```

<br>

### 💬 Majlis &nbsp;·&nbsp; [GitHub](https://github.com/alialaa932003/Majlis)

An Islamic social platform — think Twitter meets a traditional Arabic *majlis* (gathering). Posts, comments, likes, friends system, Islamic topic categories, AI caption suggestion, and AI content moderation.

My ownership: **the entire real-time layer**. Designed a JWT-authenticated Socket.IO infrastructure from scratch — singleton `SocketService` with userId→socketId mapping, typed event contracts with TypeScript generics shared across frontend and backend, and a feature-based socket handler system built for Open/Closed extensibility. Also built the full-stack notifications system: Mongoose schema, paginated REST API, real-time socket emission, and a two-layer frontend state (TanStack Query + Zustand) with zero API calls for live updates.

```
MERN · TypeScript · Socket.IO · TanStack Query · Zustand · Shadcn UI · Tailwind · Cloudinary · OpenAI
```
